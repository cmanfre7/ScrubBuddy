# ScrubBuddy

Personal productivity and tracking app for medical students (3rd/4th year clerkships).

## For AI Agents

**Read [CLAUDE.md](./CLAUDE.md) before doing any work.** It contains:
- Complete implementation status
- Known issues and solutions
- Session protocol and logging requirements
- All technical documentation

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS v4
- PostgreSQL (Railway)
- Prisma ORM
- NextAuth.js

## Deployment

Push to GitHub. Railway auto-deploys from main branch.

```bash
git push origin main
```
